API Reference

Three lines to generate a scene

Simple REST API. Generate, stream, and render.

bash 
curl -X POST https://aisketch.app/v1/sketch \
  -H "Content-Type: application/json" \
  -H "X-API-Key: sk_your_key" \
  -d '{
    "prompt": "a cozy cabin in the woods",
    "mode": "draw",
    "canvas": { "width": 800, "height": 600 }
  }'
javascript

const reader = response.body.getReader()
const decoder = new TextDecoder()

javascript 
import { renderScene, animateScene } from 'aisketch'

// Static render
const canvas = document.getElementById('canvas')
const ctx = canvas.getContext('2d')
renderScene(ctx, scene, { wobble: 0.5, fidelity: 0.7, seed: 42 })

// Animated playback ‚Äî stroke by stroke
const handle = animateScene(ctx, scene, {
  wobble: 0.5,
  fidelity: 0.7,
  seed: 42,
  strokeDuration: 400,
  layerPause: 200,
  onProgress: (p) => console.log(`${(p * 100).toFixed(0)}%`),
  onComplete: () => console.log('Done!'),
})

// Control playback
handle.pause()
handle.resume()
handle.cancel()
